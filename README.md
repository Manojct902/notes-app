# 📝 React Notes App

A simple Notes Application built using React and Tailwind CSS.
Users can add notes, view them, and delete them easily.

---

## 🚀 Features

* ➕ Add notes
* 📋 View notes
* ❌ Delete notes

---

## 🛠️ Tech Stack

* React
* JavaScript
* Tailwind CSS

---

## 📂 How It Works

* Uses `useState` to store notes
* Adds notes using form submission
* Displays notes using `.map()`
* Deletes notes using `splice()`
